# Background Music for Videos

This directory contains the background music used for generated property videos.

## default: background.mp3
The file `background.mp3` is currently a placeholder track (SoundHelix Song 1).

## How to Change the Music
To use your own music (copyright-free, luxury style):
1. Rename your MP3 file to `background.mp3`.
2. Replace the existing file in this folder.
3. The video generator will automatically use the new file.

## Where to find music?
- YouTube Audio Library (Royalty Free)
- Mixkit.co
- Pixabay Music
- Free Music Archive

Look for "Corporate", "Luxury", "Real Estate", or "Deep House" genres.
